Student Homepage Portfolio page & Student Survey Project

This project involves the creation of a Student Homepage Portfolio page and a student survey form, both hosted using Amazon Web Services. The homepage serves as a hub about the member information, while the survey form collects data from students. This initiative highlights the application of cloud computing in educational settings.

Repo: github.com

Demo: Information Portal and Survey Form

  • Tech Stack: HTML, CSS, AWS S3, Amazon EC2 , Angular2
  • Features: Homepage with navigation, error handling, survey form with data collection

Key Insights

  • Implemented using AWS services for reliability and scalability.
  • Designed to be responsive and user-friendly using Angular2 and RESTful Web services.
